PDFin button — use `primary` (violet) for the single main action per screen ("Proses", "Unduh"); `secondary` for alternates; `ghost` for quiet actions; `danger` for destructive.

```jsx
<Button variant="primary" size="lg">Gabung PDF</Button>
<Button variant="secondary">Mulai ulang</Button>
```

Never place two primary buttons on one screen. Labels are short, verb-first, sentence case.
