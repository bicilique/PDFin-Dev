# PDFin Context

## Glossary

- **Local network processing path**: The normal request and result path between a customer's internal application and its PDFin Self-hosted deployment. It describes where processing occurs; it does not imply an air-gapped environment or define outbound network behavior.
- **Customer-managed temporary storage**: Temporary storage used during a Self-hosted processing job within the customer's managed environment. Its retention, cleanup, and access controls are part of the deployment configuration.
